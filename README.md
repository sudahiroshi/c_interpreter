# c_interpreter

## パーサのURL

https://github.com/Ren-Sekiya/research

## console版のセットアップ方法

node.jsの場合

```
$ npm install
```

bunの場合

```
$ bun install
```

## console版の実行例

node.jsの場合

```
$ node c_lang.js hello.c
```

bunの場合

```
$ bun run c_lang.js hello.c
```

## Web版のセットアップ方法

```
$ npm install
$ ./node_modules_bower/bin/bower install pegjs
```
